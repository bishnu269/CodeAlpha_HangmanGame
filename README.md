# CodeAlpha_HangmanGame

## Description
A simple text-based Hangman Game developed using Python. The player guesses a hidden word one letter at a time before running out of chances.

## Features
- Random word selection
- 6 incorrect guess limit
- Win/Loss detection
- Console-based gameplay

## Technologies Used
- Python
- Random Module
